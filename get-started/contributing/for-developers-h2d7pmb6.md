
# For developers

On some stacks implementing Welcome, contribution is open to feature teams on the code side.

---

Contributions are currently open for the web vue.js stack and shared components. You can find the [documentation on GitLab](https://gitlab.softfactory-accor.net/ecom/front/applications/components/accor-core-library/-/blob/develop/CONTRIBUTING.md) (please note Zscaler access is required).

> Be warned:  
> Before submitting your contribution, please ensure the Design System team has reviewed, specified, and approved your update or new component. They will verify the [contribution model](./contribution-model/presentation.md).
