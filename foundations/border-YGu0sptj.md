
# Border

Define elements and afford interaction by applying borders on containers.

---

# Tokens

> Be warned:  
> The values shown below apply only to the legacy theme of ALL.

  
none: 0px  
default: 1px  
strong: 2px  
current: 4px  


# Guidelines

Currently, Welcome does not provide a definite guideline for border thickness. A `strong` border will appear more prominent than a `default` border.

You can apply both thicknesses for static (sometimes called “decorative”) and interactive borders.

> Some extra info:  
> Explore the [Color](./color/overview.md) section to find guidelines related to border color.
