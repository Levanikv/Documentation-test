# Component Tokens (`wel.comp`)

The component tokens are part of Welcome's component layer and specify the appearance of a dedicated component by referencing semantic color tokens.

> Be warned:  
> We limit their use to limit complexity. Only buttons consume them today

## Primary Button Colors

  
**Default**  
  
bg-color: rgb(255, 0, 208)  
border-color: rgb(255, 0, 208)  
fg-color: rgb(255, 0, 208)  
  
  
**Hover**  
  
fg-color: rgb(255, 0, 208)  
border-color: rgb(255, 0, 208)  
bg-color: rgb(255, 0, 208)  
  
  
**Pressed**  
  
fg-color: rgb(255, 0, 208)  
bg-color: rgb(255, 0, 208)  
border-color: rgb(255, 0, 208)
